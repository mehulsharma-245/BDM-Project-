# Business Data Management Project

## Overview
This repository contains my Business Data Management (BDM) project completed as part of the IIT Madras BDM program. The project focuses on applying data-driven techniques to analyze business operations and improve profitability.

## Problem Context
The project was conducted in collaboration with Om Prakash and Sons, a retail business. The objective was to identify operational challenges and suggest data-backed strategies for improving revenue and efficiency.

## Methodology
- Conducted primary data collection from the firm  
- Performed exploratory and descriptive analysis  
- Analyzed product-level performance and revenue contribution  
- Derived actionable business insights

## Tools Used
- MS Excel for data analysis and visualization  
- Business KPIs and structured analytical methods

## Key Outcomes
- Identified high and low performing products  
- Recommended optimized product assortments  
- Supported managerial decision-making with data insights
